# 🛒 E-Commerce Platform (MERN Stack)

## 📌 Project Overview
This is a full-stack **E-commerce Platform** built using the **MERN stack (MongoDB, Express, React, Node.js)**.  

Currently, the project has:
- ✅ Fully completed **Frontend (User + Seller Panel)**
- 🚧 Backend development in progress (starting with Express + MongoDB)

The platform supports:
- User shopping experience
- Seller product management
- Scalable architecture for future enhancements

---

## 🚀 Features Implemented (Frontend)

### 👤 User Panel
- Browse products
- View product details
- Add to cart
- Responsive UI design
- Navigation with React Router

### 🛍️ Seller Panel
- Add new products
- Edit existing products
- Delete products
- Manage product listings

### 🎨 UI/UX
- Modern responsive design
- Clean component-based architecture
- Reusable components

## ⚙️ Installation & Setup

### 1. Clone Repository
git clone <your-repo-url>
cd ecommerce-platform

### 2. Install Frontend Dependencies
cd client
npm install

### 3. Run Frontend
npm start

Frontend will run on:
http://localhost:3000
